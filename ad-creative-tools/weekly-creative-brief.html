Variation Rules

Entity = Format Bucket + Hook Type + Core Emotion

All three must differ from existing ads for this to count as a true new entity. Two briefs with the same format + emotion but different hooks are the same entity — you're buying one lottery ticket, not two. When in doubt, change the format first.
