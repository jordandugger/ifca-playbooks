How to Use This Audit

Pull up Meta Ads Manager, filter to your last 30–90 days of active ads. For each ad, identify which format it uses, what type of hook opens it, and what emotion it's designed to trigger. Any two ads with the same format + hook type + emotion are the same entity — the algorithm sees them as one lottery ticket. Duplicates don't expand your reach; they compete with each other for the same auction slot.
